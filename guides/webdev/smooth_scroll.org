#+TITLE: Smooth Scroll

Suppose we have the following =html= structure
#+begin_src html
<div class="container">
  <section>
    <h1>First page</h1>
  </section>
  <section>
    <h1>Second page</h1>
  </section>
  <section>
    <h1>Third page</h1>
  </section>
  <section>
    <h1>Fourth page</h1>
  </section>
</div>
#+end_src

Then to implement smooth scroll using =css=:
#+begin_src css
.container {
    scroll-snap-type: y mandatory;
    height: 100vh;
    overflow-y: scroll;
}

section {
    scroll-snap-align: start;
}
#+end_src
